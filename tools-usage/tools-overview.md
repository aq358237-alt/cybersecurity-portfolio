# 🛠️ Cybersecurity Tools Overview

This document summarizes the tools I have learned and used during my cybersecurity journey.

---

## 🌐 Network Scanning & Enumeration

### Nmap

* Used for network discovery and port scanning
* Identifies open ports and running services

### Netcat

* Used for banner grabbing and manual connections
* Can act as both client and server

### Dig

* DNS lookup tool
* Used for querying domain name information

### Whois

* Retrieves domain registration details
* Useful for reconnaissance

---

## 🌍 Web Application Security

### Burp Suite

* Intercepts and modifies HTTP requests
* Used for testing web vulnerabilities (SQLi, XSS)

### Gobuster

* Directory and file brute forcing tool
* Helps find hidden endpoints

---

## 🔐 Password Attacks

### Hydra

* Brute force login attacks
* Supports multiple protocols (SSH, FTP, HTTP)

---

## 📡 Network Analysis

### Wireshark

* Packet capturing and traffic analysis
* Used to inspect network communication

---

## 💣 Exploitation Frameworks

### Metasploit

* Framework for developing and executing exploits
* Contains payloads and exploit modules

---

## 📶 Wireless Security

### Airmon-ng

* Used for wireless network monitoring
* Enables monitor mode on wireless interfaces

---

## 🔍 Vulnerability Scanning

### Nessus

* Automated vulnerability scanner
* Identifies security issues in systems

---

## 🕵️ OSINT (Open Source Intelligence)

### OSINT Techniques

* Gathering public information about targets
* Includes domain, email, and social media analysis

Tools used:

* Whois
* Google Dorking
* Public databases

---

## 🎯 Summary

These tools help in:

* Reconnaissance
* Scanning & Enumeration
* Exploitation
* Post-exploitation analysis
